# Карта кода: файлы и функции

Справочник «что где лежит»: за что отвечает каждый файл и что делает каждая
функция. Код не дублируется — только назначение.

## Обзор файлов

| Файл | Роль |
|---|---|
| `target_fragment_analysis.py` | **Ядро расчёта**: детекция устойчивых фрагментов, спектр, коэффициенты, графики. Используется всеми интерфейсами. |
| `worker.py` | **Backend** для C++ GUI: команды `load`/`analyze` и режим `serve` (постоянный процесс). Переиспользует ядро. |
| `eeg_gui.py` | **GUI на Python** (PyQt6), самостоятельная версия с прямым вызовом ядра. |
| `cpp_gui/main.cpp` | Точка входа C++-приложения. |
| `cpp_gui/mainwindow.{h,cpp}` | Главное окно: интерфейс, логика, запуск воркера, разбор JSON. |
| `cpp_gui/signalview.{h,cpp}` | Виджет сигнала: отрисовка и выделение фрагмента мышью. |
| `cpp_gui/imageview.{h,cpp}` | Показ PNG-картинок с сохранением пропорций. |
| `cpp_gui/CMakeLists.txt` | Сборка C++-приложения (CMake + Qt6). |
| `stable_freq.py` | Ранний прототип детекции устойчивой частоты. |
| `alpha_detect.py` | Ранний прототип детекции альфа-интервалов. |
| `NP042309_CE.fif` | Пример записи ЭЭГ. |

## `target_fragment_analysis.py` — ядро расчёта

Вверху файла — константы (`DETECT_CH`, `BAND`, `ALPHA_BAND`, `ANALYSIS_BAND`,
`PSD_NFFT`, пороги детекции и др.).

| Функция | Что делает |
|---|---|
| `dominant_freq_track(sig, sfreq)` | Строит спектрограмму скользящим окном и возвращает «дорожку» доминирующей частоты во времени: `(время, частота, мощность)`. |
| `find_stable_runs(t, dom_freq, gate, min_dur, tol)` | Склеивает подряд идущие моменты в устойчивые интервалы, пока частота не уходит дальше `tol` от средней; отбрасывает короче `min_dur`. Возвращает список интервалов. |
| `detect_target_fragment(raw, sfreq)` | Главная детекция: усредняет каналы `DETECT_CH`, получает дорожку, ставит порог по мощности, находит все интервалы и выбирает самый длинный (целевой). Возвращает `(target, runs, avail)`. |
| `channel_psd(sig, sfreq)` | Спектр мощности одного канала методом Уэлча, обрезанный до полосы анализа `1–30 Гц`. Возвращает `(частоты, мощности)`. |
| `band_power(f, p, lo, hi)` | Мощность в полосе `[lo, hi]` как площадь под кривой спектра (интеграл трапециями). |
| `alpha_coefficients(f, p)` | По спектру одного канала считает все характеристики альфа-ритма (модальная частота, центроид, разброс, FWHM, добротность, альфа-индекс, отношения ритмов, пиковость, энтропия, мощности по диапазонам). Возвращает словарь. |
| `plot_topomaps(values_by_metric, info, target, fname)` | Рисует сетку топокарт (по одной на метрику) и сохраняет PNG. |
| `plot_spectra(psd_by_ch, main_ch, target, fname)` | Рисует спектры основных отведений с подсветкой альфа-полосы и сохраняет PNG. |
| `main()` | Пакетный режим: читает `.fif`, считает целевой фрагмент по всем каналам, пишет CSV в `results/` и картинки в `figures/`. Срабатывает только при прямом запуске скрипта. |

## `worker.py` — backend для C++ GUI

Словари `METRIC_KEYS` (порядок коэффициентов), `METRIC_CMAP` (цветовые схемы),
`METRIC_RU` (русские подписи для топокарты).

| Функция | Что делает |
|---|---|
| `write_result(out_dir, obj)` | Записывает объект-результат в `out_dir/result.json` (UTF-8). |
| `load_raw(path)` | Читает `.fif`, фильтрует `0.5–30 Гц`, ставит усреднённый референс. Возвращает объект записи. |
| `do_load(raw, out_dir)` | Команда «load»: каналы, sfreq, длительность, прореженный сигнал для отрисовки, список устойчивых интервалов, целевой → `result.json`. |
| `do_analyze(raw, tmin, tmax, channels_csv, metric, out_dir)` | Команда «analyze»: коэффициенты по каналам на фрагменте, `coefficients.csv`, три картинки (спектр/спектрограмма/топокарта), сводка → `result.json`. |
| `_render_spectrum(psd_by_ch, path)` | Рисует спектр выбранных каналов (PSD) в PNG. |
| `_render_spectrogram(sig, sfreq, tmin, path)` | Рисует спектрограмму (время×частота) среднего по каналам в PNG. |
| `_render_topomap(table, raw, picks, metric, path)` | Рисует топокарту выбранной метрики (если ≥4 каналов) в PNG; иначе подсказку. |
| `serve()` | Режим сервера: держит запись в памяти (`cache`), читает JSON-команды из stdin, на каждую пишет `result.json` и печатает `DONE`. Убирает повторное чтение `.fif`. |
| `get_raw(path)` (внутри `serve`) | Возвращает запись из кэша, читая её только при первом обращении к новому файлу. |
| `main()` | Разбор аргументов командной строки: `serve` / `load` / `analyze`. |

## `eeg_gui.py` — GUI на Python (PyQt6)

Класс `EEGAnalyzer(QMainWindow)` — всё окно. Расчёт вызывается напрямую из ядра,
графики рисуются через matplotlib-канвасы.

| Метод | Что делает |
|---|---|
| `__init__` | Инициализирует состояние и строит интерфейс. |
| `_build_ui` | Создаёт все виджеты: панель управления, график, вкладки результатов. |
| `_make_canvas_tab(title)` | Заводит вкладку с matplotlib-канвасом и тулбаром. Возвращает `(canvas, figure)`. |
| `_hline()` | Возвращает горизонтальную линию-разделитель. |
| `on_open()` | Диалог выбора `.fif` и загрузка. |
| `load_file(path)` | Читает запись, заполняет список каналов, детектирует интервалы, рисует сигнал, ставит SpanSelector, считает целевой фрагмент. |
| `_draw_signal()` | Рисует сигнал и подсвечивает устойчивые/целевой интервалы. |
| `_setup_span()` | Включает выделение фрагмента мышью (`SpanSelector`). |
| `_on_span_select(xmin, xmax)` | Обработчик выделения: запоминает границы фрагмента и обновляет подпись. |
| `_use_target_fragment()` | Подставляет целевой фрагмент как текущее выделение. |
| `_set_all_channels(checked)` | Отметить/снять все каналы. |
| `_select_occipital()` | Отметить только затылочно-теменные каналы. |
| `_selected_channels()` | Возвращает список отмеченных каналов. |
| `on_calc()` | Проверки и запуск расчёта по выделению. |
| `_compute(sel)` | Вырезает фрагмент, считает PSD и коэффициенты по каналам, обновляет таблицу/графики/карту. |
| `_fill_table(table)` | Заполняет таблицу коэффициентов и сводку. |
| `_plot_psd(psd_by_ch)` | Рисует спектры выбранных каналов. |
| `_plot_spectrogram(sig)` | Рисует спектрограмму фрагмента. |
| `_refresh_topomap()` | Перерисовывает топокарту под выбранную метрику. |
| `on_export()` | Сохраняет таблицу коэффициентов в CSV. |

## `cpp_gui/mainwindow.{h,cpp}` — главное окно

Структура `RunInfo` — один устойчивый интервал для таблицы фрагментов.
Свободные функции: `envOr()` (значение из env или умолчание), `metricRu()`
(ключ→русская подпись), `fitColumns()` (ширина колонок с запасом).

| Метод | Что делает |
|---|---|
| `MainWindow(parent)` | Конструктор: задаёт пути (python/проект/кэш), строит UI, запускает воркер. |
| `~MainWindow()` | Деструктор: корректно завершает процесс-воркер (`quit` → `kill` при таймауте). |
| `autoLoadDefault()` | Загружает `NP042309_CE.fif`, если он есть рядом. |
| `buildUi()` | Создаёт все виджеты, вкладки и связи (`connect`). |
| `startServer()` | Поднимает постоянный процесс `python -u worker.py serve`. |
| `sendCommand(cmd)` | Шлёт JSON-команду в stdin воркера и ждёт ответ `DONE`. |
| `readResult(out)` | Читает и парсит `result.json`; проверяет флаг `ok`. |
| `loadFile(path)` | Команда `load`: заполняет каналы, сигнал, интервалы, таблицу фрагментов; считает целевой. |
| `onOpen()` | Диалог выбора файла → `loadFile`. |
| `onCalc()` | Команда `analyze`: заполняет таблицу коэффициентов, сводку, картинки. |
| `onUseTarget()` | Ставит целевой фрагмент как выделение и считает. |
| `onSelection(tmin, tmax)` | Реакция на выделение мышью: обновляет UI. |
| `onApplyManualTime()` | Применяет фрагмент, введённый в поля времени (с проверками). |
| `onMetricChanged()` | При смене метрики перерисовывает карту (пересчёт). |
| `updateSelectionUi(tmin, tmax)` | Единая синхронизация: график + поля времени + подпись. |
| `setAllChannels(checked)` | Отметить/снять все каналы. |
| `selectOccipital()` | Отметить только затылочно-теменные каналы. |
| `selectedChannels()` | Список отмеченных каналов. |
| `onExport()` | Сохраняет последний `coefficients.csv` в выбранный путь. |
| `showError(msg)` | Показывает диалог ошибки и пишет в статус-бар. |

## `cpp_gui/signalview.{h,cpp}` — виджет сигнала

Структура `Run` — интервал для отрисовки (начало, конец, признак целевого).
Класс `SignalView : QWidget`. Сигнал `selectionChanged(tmin, tmax)`.

| Метод | Что делает |
|---|---|
| `SignalView(parent)` | Конструктор виджета. |
| `setData(t, y, runs, duration)` | Задаёт сигнал и интервалы, сбрасывает кэш фона, перерисовывает. |
| `setSelection(tmin, tmax)` | Задаёт текущее выделение (только перерисовка рамки). |
| `clearData()` | Очищает данные и выделение. |
| `buildCache()` | Рисует фон (сигнал + интервалы + оси) один раз в `QPixmap`-кэш. |
| `paintEvent()` | Рисует кэш фона, поверх — рамку выделения. |
| `resizeEvent()` | Сбрасывает кэш при изменении размера. |
| `mousePressEvent()` | Начинает выделение: запоминает время старта. |
| `mouseMoveEvent()` | Тянет границы выделения и перерисовывает. |
| `mouseReleaseEvent()` | Завершает выделение и испускает `selectionChanged`. |
| `plotRect()` | Прямоугольник области графика (с полями под подписи). |
| `xToTime(px)` / `timeToX(t)` | Перевод пиксель↔время по горизонтали. |
| `valToY(v)` | Перевод значения сигнала в координату Y. |

## `cpp_gui/imageview.{h,cpp}` — показ картинок

Класс `ImageView : QLabel`.

| Метод | Что делает |
|---|---|
| `ImageView(parent)` | Конструктор: выравнивание по центру, заглушка. |
| `setImagePath(path)` | Загружает PNG в `QPixmap` и масштабирует под размер. |

| `resizeEvent()` | Перемасштабирует картинку под новый размер. |

## Ранние прототипы (для истории)

### `stable_freq.py`
| Функция | Что делает |
|---|---|
| `dominant_freq_track(sig, sfreq)` | Дорожка доминирующей частоты (ранняя версия). |
| `find_stable_runs(...)` | Поиск устойчивых интервалов (ранняя версия). |
| `regional_power(raw, picks_eeg, s, e, f0, half, sfreq)` | Мощность в узкой полосе вокруг частоты по выбранным каналам. |
| `main()` | Прогон детекции и вывод результатов. |

### `alpha_detect.py`
| Функция | Что делает |
|---|---|
| `moving_average(x, win)` | Скользящее среднее сигнала. |
| `find_intervals(mask, times, sfreq, min_dur, merge_gap)` | Находит интервалы по булевой маске (с объединением близких и порогом длительности). |
| `main()` | Прогон детекции альфа-интервалов. |
